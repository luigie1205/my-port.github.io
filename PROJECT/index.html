Projects

Here are some of the projects I created.

Project Name: Recipe Delight

Recipe Delight is an interactive web application designed to help users find and organize recipes based on their dietary preferences, ingredients on hand, and cooking time. The application offers a seamless user experience with a visually appealing interface and responsive design, ensuring accessibility across various devices

Project Name: EventEase

EventEase is a web-based dashboard application designed to help event planners manage events efficiently. The application provides tools for scheduling, attendee management, and real-time updates, offering a user-friendly interface with intuitive navigation.

Project Name: FinanceBuddy

FinanceBuddy is a web application designed to help users track their personal finances, including income, expenses, and savings goals. The application offers an intuitive interface with visualizations to help users manage their finances effectively.
